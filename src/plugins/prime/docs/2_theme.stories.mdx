# Тема

Для удобной конфигурации темы мы используем показательные странички со всеми видами одного компонента.

Там же будут указаны гайды, как использовать компонент, чтобы получить нужный вам вид.

Базовые компоненты не будут поставляться с библиотеки, вы должны использовать праймовские компоненты, добавляя выбранный вами префикс.

Для каждого компонента будет добавлена быстрая ссылка на макет, где вы можете сверить корректно ли он выглядит в соответствии с макетом.
Макеты изначально сделаны под PrimeVue v3, мы используем v4, поэтому не все токены будут совпадать, мы самостоятельно их подгоняем
под необходимый внешний вид.

Если незнакомы с компонентом, смотрите в первую очередь вкладку Docs, там вы увидите все виды и примеры кода. Истории спроектированы в
виде таблиц чтобы показать как можно больше вариантов, в Controls добавлены все пропсы с документации, чтобы можно было потестить разные варианты.
