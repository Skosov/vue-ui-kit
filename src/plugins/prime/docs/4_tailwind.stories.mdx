# Tailwind

При желании вы можете использовать Tailwind в своих проектах. Этот гайд поможет вам подключить Tailwind в проект и сделать интеграцию с Prime.

## Установка

Для начала установите Tailwind в проект c помощью [этого гайда](https://tailwindcss.com/docs/guides/vite#vue).

Не забудьте выполнить команду `npx tailwindcss init -p` после установки зависимостей, она создает необходимые файлы для конфигурации и
подключения Tailwind.

На 4-м шаге вам необходимо добавить директивы `@tailwind`, для этого необязательно создавать новый файл, и необязательно называть его именно
`style.css`. Добавьте эти директивы в любой файл, который импортируется в `main.ts`.

## Интеграция с Prime

Для интеграции с Prime вам необходимо установить плагин `tailwindcss-primeui`.

```bash dark
yarn add tailwindcss-primeui

npm install tailwindcss-primeui
```

После установки плагина добавьте его в конфигурационный файл `tailwind.config.js`.

```js dark
export default {
  // ...
  plugins: [require('tailwindcss-primeui')],
};
```

Подробнее о плагине можно узнать в [его документации](https://primevue.org/tailwind/). Если коротко то он добавляет `primary` и `surface` цвета.

Если вы хотите использовать эти классы, чтобы переопределить стили праймовского компонента, добавляйте перед ним `!`.

```html dark
<PrimeInputText placeholder="Overriden" class="!p-8" />
```

По документации Prime предлагает использовать `layers`, чтобы не нужно было добавлять `!` перед классами, это пока сырая технология и вы убьете
стили для Chrome (как минимум) ниже 99-й версии.

### Тема

Для того чтобы все токены были доступны, нужно добавить настройки темы в `tailwind.config.js`.

```js dark
const { tailwindConfigTheme } = require('@cdek-ui-kit/vue');

export default {
  // ...
  theme: {
    extend: {
      ...tailwindConfigTheme,
    },
  },
};
```

Если вы хотите изменить или добавить свои токены в уже заданные свойства,
то можно использовать функцию `mergeTailwindTheme` из пакета `@cdek-ui-kit/vue`.

Используйте этот вариант, если вам нужно изменить `screens`, `colors`, `textColor`, `borderRadius`, `borderWidth`, `width` или `height`.

```js dark
const { mergeTailwindTheme } = require('@cdek-ui-kit/vue');

export default {
  // ...
  theme: {
    extend: {
      ...mergeTailwindTheme({
        spacing: {
          custom: '4px',
        },
      }),
    },
  },
};
```

Если ваше изменение не попадает в эти свойства, то вы можете спокойно просто добавлять свойства.

```js dark
const { tailwindConfigTheme } = require('@cdek-ui-kit/vue');

export default {
  // ...
  theme: {
    extend: {
      ...tailwindConfigTheme,
      gridTemplateColumns: {
        '100%': '100%',
      },
    },
  },
};
```

## Токены

### Медиа запросы

Медиа адаптированы под наши миксины, так что бы не было конфликтов со старым дизайном.

```js dark
screens: {
  sm: '576px',
  md: '768px',
  lg: '992px',
  xl: '1200px',
  '2xl': '1400px',
}
```

Для mobile first подхода используйте `<screen>:<rule>`, например `sm:w-10`. Для desktop first подхода используйте
`max-<screen>:<rule>`, например `max-lg:w-10`.
