# Темная тема

Темная тема была стилизована по табличке токенов. Макетов с темной темой нет, поэтому некоторые состония компонента могут отображться неправильно. При нахождении ошибки - создайте issue. В следующих релизах ошибка будет исправлена.

Для более тонкой настройки темной темы используте дополнительный конфиг на стороне проекта, как описано в разделе "Установка"
