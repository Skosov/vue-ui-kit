# Carousel
[PrimeVue Carousel](https://primevue.org/carousel/), [Макет](<https://www.figma.com/design/4TYeki0MDLhfPGJstbIicf/UI-kit-PrimeFace--DS-?node-id=1207-3811>)

### Карусель с 5-ю видимыми (из 8-ми) слайдами

```html dark
<PrimeCarousel
  :value="carouselItems"
  :numVisible="5"
  :numScroll="1"
  :responsiveOptions="responsiveOptions"
>
  <template #item="{ data }">
    <!-- Тут ваш html для каждого элемента карусели. -->
    <!-- В data лежит объект из коллекции carouselItems. -->
    <!-- Пример шаблона для слайда: -->
    <div class="flex flex-col gap-3 mx-1 px-3 py-3 border rounded">
      <span v-html="data.title" />
      <span v-html="data.subtitle" />
    </div>
  </template>
</PrimeCarousel>
```

`carouselItems` - коллекция объектов с данными для каждого элемента карусели.

```ts dark
// Пример коллекции слайдов для карусели:

const carouselSlides: Array<{ [key: string]: any }> = [
  {
    title: 'Lorem Ipsum 1',
    subtitle: 'Lorem ipsum dolor sit amet, consectetur adipisicing elit. Delectus, saepe.'
  },
  // ...
  {
    title: 'Lorem Ipsum 8',
    subtitle: 'Lorem ipsum dolor sit amet, consectetur adipisicing elit. Delectus, saepe.'
  },
];
```

`numVisible` - кол-во элементов, отображаемое за 1 прокрутку.

`numScroll (number)` - кол-во элементов, которое будет сдвигаться за 1 прокрутку.

`responsiveOptions` - настройка поведения для адаптивности.
```ts dark
// Пример опций для настройки адаптивности карусели:

import type { CarouselResponsiveOptions } from 'primevue/carousel';

const responsiveOptions: CarouselResponsiveOptions[] = [
  {
    breakpoint: '1120px',
    numVisible: 5,
    numScroll: 1,
  },
  // ...
];
```
